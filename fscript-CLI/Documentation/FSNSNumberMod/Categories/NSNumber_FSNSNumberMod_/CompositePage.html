NSNumber(FSNSNumberMod)

Extension of FSNSNumber to support the modulo operator

Extends Class: NSNumber
Declared In: FSNSNumberMod.h

Discussion

This category adds a single method, operator_percent:, that F-Script use the familiar C modulo operator, %.


Methods

-operator_percent:

operator_percent: 

- (NSNumber*) operator_percent:(NSNumber*)operand;
Discussion

This method implements modulo division between two numbers using C's % operator. The two operands are converted to integers using the intValue message before dividing. Example: result := 12 % 5. out println:result will print 3.
